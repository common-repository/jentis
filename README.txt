== Installation ==

Note: This plugin is required to install and configure WooCommerce Plugin

1. Upload the plugin files to the `/wp-content/plugins/jentis` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->Settings->Integration->JENTIS Settings screen to configure the plugin
1. Add the **Full tracking domain,** **Container ID,** **Hash** received from the JENTIS and save
